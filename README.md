# E-Commerce_Milestone_Project

**Strategic Analytics Using the Olist Brazilian E-Commerce Dataset**

Transforming raw e-commerce data into actionable business insights using Excel, MySQL, and Power BI.

---

## Project Overview

This project demonstrates an end-to-end Data Analytics workflow using the **Olist Brazilian E-Commerce Dataset**. The objective is to analyze sales performance, customer behavior, seller performance, product trends, delivery efficiency, and customer satisfaction to support data-driven business decisions. The project applies Excel, MySQL, and Power BI to clean, analyze, and visualize real-world e-commerce data. :contentReference[oaicite:0]{index=0}

---

## Objectives

- Clean and prepare raw e-commerce data.
- Import and manage relational datasets in MySQL.
- Perform SQL queries for business analysis.
- Build an interactive Power BI dashboard.
- Generate actionable insights and business recommendations.

---

## Tech Stack

- Microsoft Excel – Data Cleaning & Preparation
- MySQL – Data Storage & SQL Analysis
- Power BI – Interactive Dashboard & Data Visualization

---

## Dataset

**Dataset:** Olist Brazilian E-Commerce Dataset

The dataset contains information about:

- Customers
- Orders
- Sellers
- Products
- Payments
- Order Items
- Reviews
- Delivery Details
- Product Categories

---

## Project Workflow

### 1. Data Cleaning (Excel)

- Removed duplicate records.
- Filtered null and blank values.
- Standardized data for analysis.

### 2. SQL Database

- Created database and tables.
- Imported cleaned CSV files.
- Performed SQL operations including:
  - SELECT
  - WHERE
  - ORDER BY
  - GROUP BY
  - JOINS
  - Aggregate Functions

### 3. Power BI Dashboard

Developed interactive dashboards to analyze:

- Sales Performance
- Revenue Analysis
- Customer Insights
- Product Category Performance
- Seller Performance
- Delivery Performance
- Customer Reviews

---

## Dashboard KPIs

- Total Orders
- Total Revenue
- Average Order Value
- Average Delivery Time
- Customer Satisfaction Score
- Top Selling Categories
- Top Performing Sellers
- Monthly Sales Trend

---

## Key Insights

- Delivery delays negatively impact customer review scores.
- A small number of product categories contribute the majority of revenue.
- Repeat customers generate significant long-term business value.
- Seller performance directly influences marketplace growth.
- Interactive dashboards help identify business opportunities efficiently. :contentReference[oaicite:1]{index=1}

## Business Recommendations

### Improve Delivery Performance

Reduce delivery delays to improve customer satisfaction and increase repeat purchases.

### Invest in Top Products and Sellers

Support high-performing sellers and focus on profitable product categories.

### Increase Customer Retention

Implement loyalty programs and personalized marketing campaigns.
